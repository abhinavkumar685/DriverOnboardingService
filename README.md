# DriverOnboardingService

!! Welcome to the Driver Onboarding Service !!

This Demo Application is part of a basic Driver Onboarding Module.

System Requirements:
    This is a Maven Project on Java8 using Springboot. Ensure, Maven is installed on your system along with java8.
    It is Recommended that you use Linux Based OS.
    Install mongodb-server and use it as database while running this application.

### How to run in Local Setup ###

Change the Application Properties present in resources/application.properties according to your local mongodb-server.
After starting application, go to http://localhost:8080/swagger-ui.html#/ to trigger the apis
